# **App Name**: RIZERWEBNP

## Core Features:

- Custom Website Request Form: Allow users to submit their custom website requirements, budget, and contact details through a detailed, validated form.
- Request Notification to Discord: Automatically send all submitted request form data to a specified Discord webhook as a POST request, formatted for clear readability.
- Automated WhatsApp Redirection: Upon successful form submission, redirect the user to WhatsApp with a pre-filled, URL-encoded message containing their submission details for direct communication.
- Client-Side User Authentication: Enable users to create accounts, log in, and log out securely using the browser's local storage for session management.
- User Request Local Storage: Allow logged-in users to save, view, edit, and delete their personal website requests locally within their browser's storage.
- Website Type Gallery: Showcase various website types with images, descriptions, starting prices, and direct request buttons to inform and guide users.
- AI Design Idea Suggestion Tool: Utilize a generative AI tool to suggest creative design concepts and aesthetic directions based on user input, assisting in fleshing out the 'Design Idea' field in the request form.

## Style Guidelines:

- Primary brand color: A deep indigo (#5858B3), conveying professionalism and modern sophistication.
- Background color: A very dark, desaturated indigo (#191A23), providing a futuristic canvas for glassmorphism effects and ensuring contrast with text.
- Accent color: A bright, vibrant sky blue (#52A8ED), analogous to the primary color, for calls to action, highlights, and interactive elements, adding an energetic contrast.
- Headline font: 'Space Grotesk', a sans-serif, for a modern, techy, and futuristic appearance in titles and headers.
- Body font: 'Inter', a grotesque sans-serif, used for all body text, ensuring readability with its modern, neutral, and objective look.
- Use sleek, minimalist line icons throughout the interface, maintaining a contemporary and professional aesthetic.
- Implement a fully responsive fluid grid system optimized for all devices, utilizing card-based designs for content organization, combined with generous dark space and 'glassmorphism' effects for a modern, ethereal feel.
- Incorporate smooth transitions and subtle hover effects for interactive elements, along with tasteful loading animations, to create a fluid and engaging user experience.
